ShortStopper

Block YouTube Shorts. Fast. Local. No accounts.
Download (Chrome) View on GitHub

Install (Chrome)

  1. Download and unzip ShortStopper-chrome.zip.
  2. Open chrome://extensions and enable Developer mode.
  3. Click Load unpacked and select the unzipped folder.
Chrome does not allow “one-click install” outside the Chrome Web Store. This is the simplest unpacked install.

What it does

  • Blocks /shorts/* and /feed/shorts (redirects to Home).
  • Hides Shorts shelves, links, sidebar entries, and channel Shorts tabs.
  • Whitelist mode + per-channel overrides.
  • Local stats (no accounts, no servers).

Optional: basic ad/tracker blocking

If you enable Block ads/trackers (basic), Chrome will prompt for the <all_urls> permission. If you deny it, the blocker stays off.

iOS / Safari

iOS Safari doesn’t support Chrome extensions. This repo includes Safari content-blocker rules and a small PWA shell, but using them requires packaging an iOS content-blocker app.
