# Contributing

Thanks for helping improve ShortStopper.

## Guidelines
- Keep behavior stable unless fixing a bug or security issue.
- Prefer small, focused changes with clear rationale.
- Avoid new permissions and remote dependencies.
- Validate and sanitize any data loaded from storage or pages.

## Local checks
- Run policy checks: `node scripts/test-policy.mjs`
- Manual QA: see `README.md`
